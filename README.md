### 🎾 SayScore — Tennis Scoring & Apple Watch Companion
> *iOS · watchOS | SwiftUI · UIKit · HealthKit · Firebase*

A tennis scoring app with a **companion Apple Watch app**, featuring **voice-based scoring** using the Speech framework — so players can score hands-free mid-match.

- ⌚ Full **Apple Watch** companion app with real-time iPhone ↔ Watch data sync
- 🎙️ **Voice scoring** via Speech framework — completely hands-free
- ❤️ **HealthKit** integration for in-game health and fitness tracking
- 🔐 Auth via **Sign in with Apple**, Google Sign-In, and phone OTP
- 🔥 **Firebase** backend for auth, data storage, and real-time sync
